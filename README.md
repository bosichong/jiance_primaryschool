# 简测与复习小学知识

检验学习成果，简单测试一下。

基于JianCe应用，搭建的小学知识检测应用
JianCe是一个简简单单的学习结果检测的小工具，通过设置一组在线试题（题型包括：单选、多选、判断、简答）来检验学习结果。

题库地址：
https://bosichong.github.io/jiance_primaryschool/

### 目前题库

- [x] 小学数学概念、定义定理。
- [ ] 小学语文。
- [ ] 小学英语。

### 安装依赖

终端进入项目的根目录，先安装项目的依赖：

```sh
npm install
```

### 启动

```sh
npm run dev
```

### 编译打包

```sh
npm run build
```
这个题库是我为了测试孩子学习情况所编写的小玩意，当然也可以用来检测其他任何类型知识的学习结果的。希望大家喜欢，欢迎提交代码。
